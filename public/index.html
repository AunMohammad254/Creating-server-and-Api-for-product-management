Fashion Products API

A RESTful API for fashion products with CRUD operations

API Endpoints

GET /api/products

Returns a list of all fashion products.

Example Response:

      "rating": 4.8,
      "stock": 45,
      "brand": "Levi's",
      "category": "clothing",
      "thumbnail": "https://i.dummyjson.com/data/products/1/thumbnail.jpg",
      "images": [
        "https://i.dummyjson.com/data/products/1/1.jpg",
        "https://i.dummyjson.com/data/products/1/2.jpg",

    },
    // More products...

POST /api/products

Creates a new product. Send product data in the request body.

Request Body Example:

{
  "title": "Product Name",
  "description": "Product Description",
  "price": 99.99,
  "discountPercentage": 10,
  "rating": 4.5,
  "stock": 30,
  "brand": "Brand Name",
  "category": "Category Name",
  "thumbnail": "thumbnail-url.jpg",
  "images": ["image1-url.jpg", "image2-url.jpg"]
}

Example Response:

{
  "id": 6,
  "title": "Product Name",
  "description": "Product Description",
  "price": 99.99,
  "discountPercentage": 10,
  "rating": 4.5,
  "stock": 30,
  "brand": "Brand Name",
  "category": "Category Name",
  "thumbnail": "thumbnail-url.jpg",
  "images": ["image1-url.jpg", "image2-url.jpg"]
}

PUT /api/products/:id

Updates an existing product. Send updated product data in the request body.

Request Body Example:

{
  "title": "Updated Product Name",
  "price": 129.99
}

Example Response:

  "id": 1,
  "title": "Updated Product Name",
  "description": "A timeless denim jacket that never goes out of style",

Testing the API

You can test this API using tools like:

  • Postman
  • cURL
  • Fetch API in JavaScript
  • Axios

Try it out

Click the buttons below to test the API directly from this page. Check the browser console for results.

Response:

Example using Fetch API:

// Get all products
fetch('/api/products')
  .then(response => response.json())
  .then(data => console.log(data));

// Add a new product
fetch('/api/products', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: 'New Fashion Item',
    description: 'A trendy new fashion item',
    price: 79.99,
    brand: 'Fashion Brand',
    category: 'clothing'
  })
})
.then(response => response.json())
.then(data => console.log(data));
